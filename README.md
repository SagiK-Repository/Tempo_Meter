# Guitar_BPM_Detector
> Guitar BPM Detector (여러 악기의 BPM을 실시간으로 알 수 있다)

![image](https://user-images.githubusercontent.com/66783849/195984076-750844b9-2ef9-4e93-894d-7c5efb68a524.png)
